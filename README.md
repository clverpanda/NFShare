# NFShare
### Share everything with your friends through your Android smartphone.
该项目是我的毕设项目，具体内容如下：
> 本课题拟研究通过运用现代智能手机的NFC,WIFI等通讯技术实现智能手机之间近距离的资源共享。运用一种或者多种通讯技术使得智能手机之间共享资源的方式更加便捷，更加可靠。同时也要确保资源共享的安全性。
